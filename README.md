# parseja
Parsing Japanese text

Формулы для поиска в интернете понятия X на японском языке:

- Xとは (что есть X)
- Xとは何 (что такое X)
- Xとは何か (что такое X?)
- Xとは何ですか (что такое X?)
- Xとは何だろう (что же такое X?)
- Xの定義 (определение X)
- Xの定義は (определение X есть...)
- Xの意味 (значение X)
- Xの意味は (значение X есть...)
- Xについて (о X, относительно X)
- Xを説明 (объяснить X)
- Xの説明 (объяснение X)
- Xとは何を意味するか (что означает X)
- Xって何 (а что такое X? разговорная форма)
- Xってなに (что же такое X? разговорная форма)
- Xをわかりやすく (X простыми словами)

Японские соответствия выбранных терминов:

- Мышление — 思考 (shikō)
- Понимание — 理解 (rikai)
- Система — システム (shisutemu) или 体系 (taikei)
- Государство — 国家 (kokka)
- Наука — 科学 (kagaku)
- Прогноз — 予測 (yosoku)
- Угроза — 脅威 (kyōi)
- Техника — 技術 (gijutsu)
- Элемент — 要素 (yōso)
- Эксплуатация — 搾取 (sakushu) [человека человеком] или 運用 (un'yō) [об оборудовании]
